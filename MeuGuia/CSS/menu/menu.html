Dropdown

É um componente que permite exibir um menu suspenso com opções quando um elemento específico, como um botão ou link, é clicado ou passado o mouse sobre ele. É uma forma comum de fornecer uma lista de opções para os usuários selecionarem em uma interface.

    
nav ul {
      list-style: none;
      padding: 0;
      margin: 0;
    }

    nav li {
      display: inline-block;
      position: relative;
    }

    nav a {
      display: block;
      padding: 8px 16px;
      text-decoration: none;
      color: #333;
    }

    nav .dropdown-content {
      display: none;
      position: absolute;
      background-color: #f9f9f9;
      min-width: 160px;
      box-shadow: 0px 8px 16px 0px rgba(0, 0, 0, 0.2);
      padding: 8px;
      z-index: 1;
    }

    nav li:hover .dropdown-content {
      display: block;
    }      
    
  
    

<nav>
  <ul>
    <li>
      <a href="#">Home</a>
    </li>
    <li>
      <a href="#">Serviços</a>
      <div class="dropdown-content">
        <a href="#">Opção 1</a>
        <a href="#">Opção 2</a>
        <a href="#">Opção 3</a>
      </div>
    </li>
    <li>

    </li>
  </ul>
</nav>
