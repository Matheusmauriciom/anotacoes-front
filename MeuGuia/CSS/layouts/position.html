CSS Position

A propriedade Position ajuda a manipular a localização / posicionamentodo do elemento na minha página.

  • position: static; (padrão)
  • position: relative;
  • position: absolute;
  • position: fixed; (fixo a página)
  • position: sticky; (ao usar o scroll o elemento acompanha). top:0 obrigatorio
  • z-index

top, right, bottom e left.

Relative

Ao aplicar position: relative a um elemento, ele permanecerá dentro do fluxo normal da página, mas você pode ajustar sua posição usando as propriedades top, right, bottom e left.

Ele é o "pai" do absolute, referência

Absolute

Permite posicionar um elemento livremente em relação ao seu elemento pai mais próximo com uma posição definida.

    
    <div class="container">
      <img src="#" alt="#">
      <h3>Exemplo</h3>
    </div>
      
    
      
        .container{
          width: 300px;
          height: 300px
          postion: relative;
        }

        .container img{
          width: 100%;
          height: 100%;
          object-fit: cover;
        }

        .container h3{
          font-size: 40px;
          font-wight: 500;
          margin: 0;
          postion: absolute;
          top: 50%;
          left: 10%;
        }

z-index

A propriedade CSS z-index é usada para controlar a ordem de empilhamento dos elementos posicionados, ou seja, determina qual elemento deve aparecer acima ou abaixo dos outros quando há sobreposição. Ela é aplicada a elementos que tenham uma posição definida (position: absolute, position: relative ou position: fixed).

Em resumo, o z-index é usado para controlar a ordem de empilhamento dos elementos posicionados, permitindo que você determine quais elementos devem aparecer acima ou abaixo dos outros quando há sobreposição. É uma propriedade útil para controlar a aparência visual e a camada de elementos em um layout.
