Efeitos gradiente

Existem dois tipos principais de gradientes CSS: gradientes lineares e gradientes radiais.Tanto a propriedade background quanto a propriedade background-image podem ser usadas para aplicar gradientes.

Linear

Um gradiente linear cria uma transição de cor ao longo de uma linha reta.Você pode definir a direção do gradiente usando, to top to right to bottom right entre outros. 

    
      /* Gradiente linear de cima para baixo  */
.gradient-top-to-bottom {
  background: linear-gradient(to bottom, red, blue);
}
    
  
    
      /* Gradiente linear da esquerda para a direita */ 
.gradient-left-to-right {
  background: linear-gradient(45deg to right, orange, purple);
}
    
  
    
      /* Gradiente linear da parte inferior esquerda para a parte superior direita */
.gradient-bottom-left-to-top-right {
  background: linear-gradient(to top right, green, yellow);
}
    
  
    
      /* Gradiente linear da parte superior esquerda para a parte inferior direita */
.gradient-top-left-to-bottom-right {
  background: linear-gradient(to bottom right, pink, teal);
}

Gradients Radiais

Um gradiente radial cria uma transição de cor a partir de um ponto central para o exterior em todas as direções. Você pode especificar a forma do gradiente usando circle ellipse ou farthest-corner

    
  /* Gradiente radial circular */    
.gradient-circle {
  background: radial-gradient(circle, yellow, orange);
}
    
  
    
 /*  Gradiente radial elíptico */
.gradient-ellipse {
  background: radial-gradient(ellipse, cyan, purple);
}
    
  
    
/* Gradiente radial a partir do canto superior esquerdo */
.gradient-corner {
  background: radial-gradient(at top left, pink, blue);
}
    
  
    
 /* Gradiente radial a partir do ponto mais distante */
.gradient-farthest {
  background: radial-gradient(farthest-corner, red, green);
}

Opacity

Frequentemente usada em conjunto com o pseudo-seletor hover para criar efeitos de transparência em imagens e botões.

    
    .img-opacity {
      opacity: 1;
      transition: opacity 0.3s ease;
    }

    .img-opacity:hover {
      opacity: 0.5;
    }

    .btn-opacity {
      opacity: 1;
      transition: opacity 0.3s ease;
    }

    .btn-opacity:hover {
      opacity: 0.7;
    }
