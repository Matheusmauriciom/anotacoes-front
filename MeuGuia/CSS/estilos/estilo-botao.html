Estilização de Botões.

Esses elementos são amplamente utilizados em interfaces web, portanto, entender como estilizá-los de forma eficaz e atraente é importante.

Estilização de Botões

  • Personalização de estilos para botões, incluindo cores, tamanhos e efeitos de hover
  • Criação de botões com diferentes estilos, como botões de call-to-action ou botões de alternância
  • Utilização de pseudo-classes para estilizar botões em diferentes estados

Botão na vertical

    
.box {
    display: flex;
    align-items: center;
    justify-content: center;
    border: 1px solid #ccc;
    height: 300px;
}

.box__content {
    display: flex;
    flex-direction: column;
    align-items: center;
}

.btn {
    padding: 10px;
    background-color: #ccc;
    color: #fff;
    text-decoration: none;
    margin-top: 10px;
}

Estilização de links

    
      seletor específico:
      a.botao1{
        background: red;
        color: white;
        padding: 10px;
        text-decoration: none;
        border: 2px solid black;
        border-radius:5px;
      }

Hover

    
      a.botao1:hover{
        background: white;
        color: black;
        padding: 12px;
      }
