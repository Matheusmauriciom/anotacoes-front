Estilização de Cards

Esses elementos são amplamente utilizados em interfaces web, portanto, entender como estilizá-los de forma eficaz e atraente é importante.

Estilização de Cards

  • Criação de cards com diferentes estilos, como cartões de produtos, cartões de perfil ou cartões de artigos
  • Personalização de layout, cores e efeitos de transição
  • Utilização de técnicas de posicionamento e alinhamento para criar grids de cards responsivos

Texto truncado

O texto truncado é uma técnica que consiste em exibir apenas uma parte do texto em um elemento, ocultando o restante que excede o espaço disponível. Isso é útil em situações onde o espaço é limitado e deseja-se fornecer uma indicação visual de que há mais informação além do que está sendo mostrado.

A técnica é comumente implementada utilizando as propriedades white-space: nowrap, overflow: hidden e text-overflow: ellipsis.

    
.container {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
.card{
  height: auto;
  width: 40%;
  background: white;
  text-aling: center;
}

.card:hover .container{
  white-space: normal;
}

box-shadow

Ela permite controlar a aparência da sombra, como cor, desfoque, deslocamento horizontal e vertical e o tamanho do espalhamento.

horizontal-offset define o deslocamento horizontal da sombra em relação ao elemento.

vertical-offset define o deslocamento vertical da sombra em relação ao elemento.

blur-radius> define o desfoque da sombra. Quanto maior o valor, mais suave será a sombra.

spread-radius define o tamanho do espalhamento da sombra. Valores positivos aumentam o tamanho e valores negativos diminuem.

color define a cor da sombra.

    
       /* Exemplo padrão de card */
.box {
  width: 40%;
  height: 300px;
  background-color: white;
  box-shadow: 0 4px 8px 0 rgba(0,0,0,0.2), 0 6px 20px 0 rgba(0,0,0,0.19);
}
