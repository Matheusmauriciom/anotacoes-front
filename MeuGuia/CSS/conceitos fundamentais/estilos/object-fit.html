Object-fit

O object-fit é uma propriedade do CSS que permite controlar como uma imagem ou um vídeo se encaixa dentro de seu contêiner. Ele determina o dimensionamento e o posicionamento do conteúdo visual, ajudando a controlar o layout de forma mais precisa.

    
      .div{
        width: 400px;
        height: 300px;
        border: solid 5px blue;    
    }

object-fit: fill; O conteúdo é dimensionado para preencher completamente o contêiner, podendo distorcer a proporção original da imagem ou do vídeo.

object-fit: contain; O conteúdo é dimensionado proporcionalmente para se ajustar completamente dentro do contêiner, mantendo sua proporção original. Pode haver espaços em branco ao redor do conteúdo se o contêiner tiver uma proporção diferente.

object-fit: cover; O conteúdo é dimensionado proporcionalmente para cobrir completamente o contêiner, mantendo sua proporção original. Pode haver um corte do conteúdo se o contêiner tiver uma proporção diferente.

object-fit: none; O conteúdo mantém sua proporção original e não é dimensionado de forma alguma. Ele pode transbordar do contêiner, se necessário.

object-fit: scale-down O conteúdo é dimensionado para caber no contêiner, mas se o conteúdo for menor que o contêiner, ele será dimensionado como none. Caso contrário, ele será dimensionado como contain.

teste 
      
        #teste{
          width: 100%;
            height: 100%;
            object-fit: contain;
            object-position: left;
        }
