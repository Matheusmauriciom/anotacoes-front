Seletores Avançados

Aprofunde-se nos seletores CSS para segmentar elementos específicos e aplicar estilos de forma seletiva. Domine os seletores de classe, id, descendentes, filhos, irmãos, pseudo-classes e pseudo-elementos.

  • Estilização de pseudoclasses, como :hover e :focus
  • Estilização de pseudoelementos, como ::before e ::after

Psedo-classes

:first-child/ :last-child / :nth-child

A pseudo-classe :first-child seleciona o primeiro elemento filho de seu elemento pai, independentemente do tipo desse elemento. Ela é aplicada ao primeiro filho de seu pai, não levando em consideração seus irmãos.

A pseudo-classe :last-child seleciona o último elemento filho de seu elemento pai, independentemente do tipo desse elemento. Ela é aplicada ao último filho de seu pai, não levando em consideração seus irmãos.

A pseudo-classe :nth-child seleciona um elemento com base em sua posição em relação aos seus irmãos do mesmo tipo. Você pode usar um número, uma expressão matemática ou palavras-chave como "odd" (ímpar) ou "even" (par) para definir o padrão de seleção. 

    
p:first-child{
color: red;
}
p:last-child {
color: red;
} 
p:nth-child(4){
background-color: black;
}

:first-of-type/ :last-of-type / :nth-of-type:

pseudo-classe :first-of-type seleciona o primeiro elemento de um determinado tipo dentro do seu elemento pai. Ela é aplicada ao primeiro elemento do tipo especificado, independentemente de sua posição em relação a outros elementos do mesmo tipo.

A pseudo-classe :last-of-type seleciona o último elemento de um determinado tipo dentro do seu elemento pai. Ela é aplicada ao último elemento do tipo especificado, independentemente de sua posição em relação a outros elementos do mesmo tipo.

A pseudo-classe :nth-of-type seleciona um elemento com base em sua posição em relação aos seus irmãos do mesmo tipo dentro do seu elemento pai. Assim como :nth-child, você pode usar um número, uma expressão matemática ou palavras-chave como "odd" (ímpar) ou "even" (par) para definir o padrão de seleção. 

        
/* sempre vou pegar, independente */
p:first-of-type{
    background-color: green;
}
p:last-of-type{
    background-color: gray;
}
p:nth-of-type(2){
    background-color: black;
}
