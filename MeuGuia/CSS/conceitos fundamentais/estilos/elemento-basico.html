Elementos Básicos

box model

Compreender o Box Model é fundamental, Ele define como os elementos HTML são renderizados como caixas, incluindo as propriedades de margem, borda, preenchimento e conteúdo.

Background

Cor de fundo

O background-color é uma propriedade especifica a cor de fundo de um elemento.

body { background-color: lightblue; }

Background-image

O background-image é uma propriedade especifica uma imagem a ser usada como plano de fundo de um elemento.Por padrão, a imagem é repetida para cobrir todo o elemento.

  
    background-color: gray url("../img/exemplo.jpg") no-repeat center;
    background-size:cover;

Propriedades abreviadas:
background-color
background-image
background-repeat
background-attachment
background-position
Não importa se um dos valores da propriedade está faltando, desde que os outros estejam nesta ordem

Fonts

Shorthand você pode usar a propriedade font para definir várias características da fonte em uma única declaração. 

    
 font: [font-style] [font-variant] [font-weight] [font-size]/[line-height] [font-family];

 font: [font-weight] [font-size] [font-family];
 p {
  font: bold 16px Arial, sans-serif;
}


font-style: Define o estilo da fonte (normal, itálico ou oblíquo).
font-variant: Define a variante da fonte (normal ou small-caps).
font-weight: Define a espessura da fonte (normal, negrito ou valores numéricos).
font-size: Define o tamanho da fonte (valor numérico seguido de uma unidade de medida).
line-height: Define a altura da linha (valor numérico seguido de uma unidade de medida ou um valor adimensional).
font-family: Define a família da fonte (nome da fonte ou uma lista de fontes em ordem de preferência).
