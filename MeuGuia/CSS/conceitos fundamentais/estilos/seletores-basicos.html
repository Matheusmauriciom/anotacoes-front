Seletores básicos

  • Introdução aos seletores CSS básicos.
  • Aplicação de estilos a elementos específicos.
  • Utilização de classes e IDs para seleção e estilização.

Seletores e classe

Seletor de elemento: Seleciona elementos com base em seu nome de tag.

p { color: blue; }

Seletor de classe

Seletor de classe: Seleciona elementos com base no valor do atributo class.

<p class="destaque">Este é um parágrafo destacado.</p>

.destaque { font-weight: bold; color: red; }

Nesse exemplo, o parágrafo com a classe "destaque" teria o texto em negrito e na cor vermelha

Seletores de ID

Seleciona um elemento específico com base no valor do atributo id. O valor do ID deve ser único na página, ou seja, não pode haver outros elementos com o mesmo id

<p id="meu-paragrafo">Este é o meu parágrafo.</p>

#meu-paragrafo { background-color: yellow; }

Nesse exemplo, o parágrafo com o ID "meu-paragrafo" teria um fundo amarelo.

Especificidade

É uma forma de determinar quais estilos serão aplicados a um elemento quando várias regras de estilo conflitam.

    
    /* 1 - Seletor de ID (#myElement) */
    #myElement {
      color: green;
    }

    /* 2 - Tag + class  */
    tag.myClass {
      color: blue;
    }

    /* 3 - Seletor de classe (.myClass) */
    .myClass {
      color: orange;
    }

    /* Regra 4 - Seletor de elemento (p) */
    p {
      color: red;
    }

1. Seletor de tipo:

p { /* Estilos aplicados a todos os elementos p (parágrafos) */ }

2. Seletor de classe:

.destaque { /* Estilos aplicados a todos os elementos com a classe "destaque" */ }

3. Seletor de ID

#cabecalho { /* Estilos aplicados a um elemento com o ID "cabecalho" */ }

4. Seletor de descendência:

div p { /* Estilos aplicados a todos os elementos p que estão dentro de um elemento div */ }

5 .Seletor de filho direto:

ul > li { /* Estilos aplicados a todos os elementos li que são filhos diretos de uma lista não ordenada ul */ }

6. Seletor de irmão adjacente:

h2 + p { /* Estilos aplicados a todos os elementos p que são irmãos adjacentes de um elemento h2 */ }

7. Seletor de irmãos subsequentes:

h2 ~ p { /* Estilos aplicados a todos os elementos p que são irmãos subsequentes de um elemento h2 */ }

8. Seletor :not

p:not(.destaque) { /* Estilos aplicados a todos os elementos p que NÃO têm a classe "destaque" */ }

Seletores de atributo

Seletor de atributo de início

Ele é usado para selecionar elementos que possuem um determinado atributo e cujo valor começa com um determinado texto específico.

atributo: É o nome do atributo que você deseja selecionar.

^= É o operador que indica que o valor do atributo deve começar com o texto especificado.

"valor"É o valor que você deseja que o atributo comece.

a[href^="https://"] 

<a href="https://www.exemplo.com">Exemplo</a>
<a href="https://www.outroexemplo.com">Outro Exemplo</a>
<a href="http://www.nao-sera-selecionado.com">Não Será Selecionado</a>

No exemplo acima, o seletor a[href^="https://"] selecionará os dois primeiros links, pois seus atributos href começam com "https://".

O terceiro link não será selecionado, pois seu atributo href começa com "http://".

O seletor de atributo de início é útil quando você deseja selecionar elementos com base nos valores iniciais de seus atributos. Ele permite que você aplique estilos específicos apenas a elementos que atendam a esse critério.

a[href="https://www.exemplo.com"] { /* Estilos aplicados a links com o atributo href definido como "https://www.exemplo.com" */ }

Seletor de atributo de final

Ele é usado para selecionar elementos que possuem um determinado atributo e cujo valor termina com um determinado texto específico.

atributo: É o nome do atributo que você deseja selecionar.

$= É o operador que indica que o valor do atributo deve terminar com o texto especificado.

"valor" É o valor que você deseja que o atributo termine.

img[src$=".png"] 

<img src="imagem1.png" alt="Imagem 1">
<img src="imagem2.jpg" alt="Imagem 2">
<img src="imagem3.png" alt="Imagem 3">

No exemplo acima, o seletor img[src$=".png"] selecionará as imagens "imagem1.png" e "imagem3.png", pois seus atributos src terminam com ".png"

A imagem "imagem2.jpg" não será selecionada, pois seu atributo src não termina com ".png".

O seletor de atributo de final é útil quando você deseja selecionar elementos com base nos valores finais de seus atributos. Ele permite que você aplique estilos específicos apenas a elementos que atendam a esse critério.
