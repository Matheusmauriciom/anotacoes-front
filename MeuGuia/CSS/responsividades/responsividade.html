Media Queries, Unidades de Medida Responsivas, Mobile-first

A capacidade de criar layouts responsivos é uma habilidade crucial, considerando a variedade de dispositivos e tamanhos de tela em uso atualmente.

Aprender a utilizar media queries e técnicas de responsividade ajudará a criar designs que se adaptam às diferentes resoluções e tamanhos de tela.

Media Queries

Entenda como usar media queries para aplicar estilos diferentes em diferentes tamanhos de tela e dispositivos. Domine a definição de breakpoints e a aplicação de estilos específicos em cada breakpoint.

Utilização de media queries para criar layouts responsivos e adaptáveis a diferentes dispositivos

Adaptação do design e estilos para telas menores e diferentes orientações

    
      /* Smartphones (600px para baixo) */
@media only screen and (max-width: 600px) {...}

/* Dispositivos um pouco maiores, mas ainda pequenos (Pequenos Tablets e Smartphones + largos, 600px para cima) */
@media only screen and (min-width: 600px) {...}

/*Dispostivos Médios (Tablets deitados, 768px para cima) */
@media only screen and (min-width: 768px) {...}

/* Dispostivos Largos (laptops/desktops, 992px para cima) */
@media only screen and (min-width: 992px) {...}

/* Super Largos (Telas maiores laptops, desktops e até TVs, 1200px para cima) */
@media only screen and (min-width: 1200px) {...}

Unidades de Medida Responsivas

Familiarize-se com unidades de medida responsivas, como porcentagem (%), viewport width (vw), viewport height (vh) e em (em). Essas unidades ajudam a criar layouts flexíveis e adaptáveis.

Mobile-first

Utilização de técnicas de responsividade para melhorar a experiência do usuário em dispositivos móveis.

Adote a abordagem mobile-first ao projetar e desenvolver seus layouts, priorizando a experiência móvel e expandindo para telas maiores

Certifique-se de que seu site seja bem otimizado para dispositivos móveis.

Overflow

É uma propriedade que controla como o conteúdo de um elemento é tratado quando excede o espaço disponível dentro desse elemento.

    
.container {
      width: 200px;
      height: 100px;
      border: 1px solid black;
      overflow: scroll;
    }

    overflow: visable (padrão)
    overflow: visible O conteúdo que excede o espaço disponível será exibido fora dos limites do elemento.    
    overflow: hidden O conteúdo excedente é cortado e não será visível.
    overflow: scroll Barras de rolagem serão adicionadas automaticamente ao elemento para permitir a visualização do conteúdo excedente. 
    overflow: auto semelhante ao scroll no entanto, se o conteúdo não exceder o espaço disponível, as barras de rolagem não serão exibidas.
