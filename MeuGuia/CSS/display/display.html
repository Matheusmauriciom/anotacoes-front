Display block/inline/inline-block

Block

A propriedade display: block é utilizada para definir um elemento como um bloco. Isso significa que o elemento ocupa toda a largura disponível e sempre inicia em uma nova linha.

Alguns exemplos de elementos que são exibidos como blocos são:

<h1></h6> <p> <div> <form> <ul>< <li> <header> <section> <footer> 

    
div {
  display: block;
}

Inline

A propriedade display: inline é usada para definir um elemento como uma linha de texto. O elemento é renderizado sem quebra de linha, de forma que outros elementos possam ficar ao seu lado. Alguns exemplos de elementos exibidos como inline são:

<a><a> <span><span> <img><img>

    
span {
  display: inline;
}

Inline-block

A propriedade display: inline-block combina características do inline e block. O elemento é renderizado como uma linha de texto, mas também pode ter largura e altura definidas, além de permitir margens e preenchimento. Essa propriedade é útil quando você deseja ter um elemento que possa ser estilizado como um bloco, mas ainda assim se comporte como uma linha de texto.

    
button {
  display: inline-block;
  width: 120px;
  height: 40px;
  background-color: blue;
  color: white;
  border: none;
  text-align: center;
}
