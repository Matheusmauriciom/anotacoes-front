Iframes

Um <iframe> HTML é usado para exibir uma página da web dentro de uma página da web.

Usar os atributos height e width para especificar o tamanho do iframe.

Boa prática sempre incluir title usado para leitores de tela.

Use border:none; para remover a borda ao redor do iframe.

<iframe src="" width="600" height="450" title="Mapa" style="border:0;"></iframe>
