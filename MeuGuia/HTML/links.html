Links e âncora

Criação de links para outras páginas e recursos externos

Utilização de âncoras para navegação interna em uma página

Exploração de atributos de link, como target e rel

Links

        
Link absoluto(URL)

        
          <a href="https://google.com">google</a>
          <a href="https://google.com" target="_blank">Ir para o site</a>
        
      
      
Link relativo

      
        <a href="pasta/index.html">Ir para index dentro da pasta</a>
        <a href="../pasta/index.html">Volta uma pastas</a>
        <a href="../../pasta/index.html">Volta duas pastas</a>
