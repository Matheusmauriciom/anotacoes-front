document.getElementById()

O método getElementById() retorna um único elemento que possui o atributo "id" correspondente ao valor fornecido como argumento.

      
      const element = document.getElementById('myElement')
      
      <h1 id="myElement">Matheus Mauricio</h1>

document.querySelector

O método querySelector() retorna o primeiro elemento que corresponde ao seletor CSS especificado como argumento.

        
          document.querySelector(#myElement)
        
         <h1 id="myElement">Matheus Mauricio</h1>

document.querySelectorAll()

O método querySelectorAll() retorna uma NodeList contendo todos os elementos que correspondem ao seletor CSS especificado como argumento.

          
            document.querySelectorAll(".myClass")[0]

getElementsByClassName()

O método getElementsByClassName()retorna uma coleção de elementos que possuem a classe CSS correspondente ao valor fornecido como argumento

        
          //Passar o indice []
          var elements = document.getElementsByClassName('myClass')[0,1,2];

Propriedades

textContent lida apenas com texto simples, enquanto innerText interpreta tags HTML, mas não exibe elementos ocultos. Já innerHTML interpreta e renderiza o conteúdo HTML completo de um elemento

Textcontent

A propriedade textContent é usada para obter ou definir o texto puro de um elemento, sem interpretar ou renderizar tags HTML dentro dele. 

    
var elemento = document.getElementById('meuElemento');
var texto = elemento.textContent; // Obtém o texto do elemento
elemento.textContent = 'Novo texto'; // Define um novo texto para o elemento

InnerText

A propriedade innerText é semelhante à textContent, mas tem uma diferença importante. Ela interpreta e renderiza as tags HTML, mas não exibe elementos ocultos por CSS. Isso significa que se um elemento estiver definido como display: none no CSS, o innerText não o exibirá

    
var elemento = document.getElementById('meuElemento');
var texto = elemento.innerText; // Obtém o texto interpretado do elemento
elemento.innerText = 'Novo texto'; // Define um novo texto para o elemento

innerHTML

A propriedade innerHTML é usada para obter ou definir o conteúdo HTML de um elemento, incluindo todas as tags HTML contidas nele. Isso permite que você insira ou modifique elementos HTML, além de texto simples. No entanto, tenha cuidado ao usar essa propriedade, pois ela pode expor seu código a vulnerabilidades de segurança, como ataques de script entre sites (XSS). Portanto, certifique-se de sanitizar ou validar o conteúdo antes de inseri-lo no innerHTML

    
var elemento = document.getElementById('meuElemento');
var conteudoHTML = elemento.innerHTML; // Obtém o conteúdo HTML do elemento
elemento.innerHTML = '
Novo conteúdo
'; // Define um novo conteúdo HTML para o elemento

.value

É usada para obter o valor de um campo de formulário em JavaScript. Ela é uma propriedade específica para campos de formulário, como input select e textarea, e permite acessar o valor inserido ou selecionado pelo usuário.

    
var campo = document.getElementById('meuCampo'); // Substitua 'meuCampo' pelo ID do seu campo de formulário
var valor = campo.value; // Obtém o valor do campo
console.log(valor); // Exibe o valor do campo no console

Exemplos

Exemplo cadastro email

quando o usuário preenche o campo de e-mail e clica no botão "Enviar", a função cadastrarEmail() é acionada, obtendo o valor do e-mail digitado e exibindo uma mensagem de feedback na div "newsletterFeedback" informando que o e-mail foi cadastrado com sucesso.

<label for="txtEmail">Email:</label>
<input type="email" id="txtEmail" autocomplete="off"/>

<button onclick="cadastrarEmail()" id="btn">Enviar</button>
<div id="newsletterFeedback"> </div>
    
// formulario
const txtEmail = document.getElementById("txtEmail")
const msg = document.getElementById("newsletterFeedback");

//funcao button
function cadastrarEmail (){
let email = txtEmail.value;
msg.innerHTML = `o email ${email} foi cadastrado com sucesso`
}

Habilitar ou desabilitar um input

O campo de entrada de e-mail está desabilitado (disabled) e exibe o valor "matheus@server.com".

Quando o botão é clicado, a função editarEmail() é executada, habilitando o campo de entrada e dando foco a ele, permitindo ao usuário editar o valor.

Quando o campo perde o foco, a função disableEmail() é chamada, desabilitando novamente o campo de entrada e tornando-o não editável.


<label for="txtEmail">Email:</label>
<input
  type="email"
  id="txtEmail"
  autocomplete="off"
  disabled
  value="matheus@server.com"
  onblur="disableEmail()"
/>
<button onclick="editarEmail()" id="btn"></button>

  
const txtEmail = document.getElementById("txtEmail")
function editarEmail() {
    txtEmail.disabled = false
    txtEmail.focus()
}

function disableEmail(){
    txtEmail.disabled= true
}

Manipulação da propriedade disabled para habilitar ou desabilitar um botão com base no estado do checkbox


<input type="checkbox" id="contrato" onchange="enableOrDisableButton()" />
<label for="contrato">Li e aceito o seguinte contrato</label>
<textarea name="" id="" cols="30" rows="10" readonly>
  Este contrato nos dá direito à ...
</textarea>
<input id="enable" type="submit" value="cadastrar" disabled />

  
 function enableOrDisableButton() {
const checkbox = document.getElementById("contrato");
const enable = document.getElementById("enable");

if (checkbox.checked) {
enable.disabled = false;
} else {
enable.disabled = true;
}
 } 

  /*enable.disabled = !checkbox.checked;
 operador de negação (!) para inverter o valor do checkbox.checked, e o resultado é atribuído à propriedade disabled do botão.*/
