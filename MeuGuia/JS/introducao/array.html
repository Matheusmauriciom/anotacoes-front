Array [ ]

Arrays em JavaScript são utilizados para armazenar múltiplos valores em uma única variável. Eles são uma coleção ordenada de elementos, acessados por meio de índices numéricos. Diferentemente dos objetos, que utilizam propriedades com nomes específicos atribuídos aos seus valores, os arrays não possuem propriedades. Os elementos dentro de um array são identificados e acessados pela sua posição indexada, começando a partir do índice 0.

  • Utilização de arrays para armazenar e manipular conjuntos de dados
  • Utilização de loops (for, while) para percorrer e processar elementos de um array
  • Criação e manipulação de arrays em JavaScript
  • Utilização de métodos de array, como push, pop, splice, map, filter e reduce
  • Iteração sobre os elementos de um array usando loops

Iterar sobre uma array de objetos

      
    function exemploIterarArray() {
      const pessoas = [
        { nome: "Ana", idade: 29, cidade: "São Paulo" },
        { nome: "Carlos", idade: 35, cidade: "Rio de Janeiro" },
        { nome: "Eduarda", idade: 22, cidade: "Belo Horizonte" }
      ];
    
      console.log(`${pessoas[0].nome} - ${pessoas[0].cidade}`); // Saída: "Ana - São Paulo"
      console.log(pessoas[1].cidade); // Saída: "Rio de Janeiro"
    
      // Iteração comum
      for (let i = 0; i < pessoas.length; i++) {
        console.log(`Seu nome é: ${pessoas[i].nome}, ${pessoas[i].idade}, reside em ${pessoas[i].cidade}`);
      }
    }

Metodos arrays

callback: Uma função que será executada em cada elemento do array.

elemento: O elemento atual sendo processado no array

índice (opcional): O índice do elemento atual sendo processado no array.

array (opcional): O array em que o método map() foi chamado.

thisArg (opcional): Um valor para ser usado como this ao executar a função callback.

Esses métodos every() e some() são úteis para fazer verificações e tomar decisões com base nos elementos de um array.

every

O método every() verifica se todos os elementos de um array atendem a uma determinada condição. Ele retorna true se a condição for verdadeira para todos os elementos e false se pelo menos um elemento não atender à condição. 

        
const arr = [1, 5 ,10, "ola", true]
let numeros = arr.every(function (elemento){
    return typeof elemento === "number"
})

console.log(numeros) // false

some

O método some() retorna true se a condição for verdadeira para pelo menos um elemento e false se nenhum elemento atender à condição. 

        
const arr = [1, 5 ,10, "ola", true]
let numeros = arr.some(function (elemento){
    return typeof elemento === "number"
})

console.log(numeros) // true

Os métodos forEach(), filter() map(). São três métodos muito úteis para manipulação de arrays em JavaScript.

forEach

O método forEach() é usado para executar uma função em cada elemento de um array. Ele não retorna um novo array, apenas itera sobre os elementos existentes.

array.forEach(callback(elemento, índice, array), thisArg) 
  
const numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(number) {
  console.log(number); //O resultado será a impressão dos números de 1 a 5.
});

filter

O método filter() cria um novo array contendo apenas os elementos que atendem a uma determinada condição. Ele retorna um array com os elementos filtrados.

array.filter(callback(elemento, índice, array), thisArg) 
  
  const numbers = [1, 2, 3, 4, 5, "a", true];

const evenNumbers = numbers.filter(function (number) {
  return number % 2 === 0;
});
// return typeof number === "number"; // Output: [1,2,3,4,5]
console.log(evenNumbers); // Output: [2, 4]

map

O método map() cria um novo array com base em uma transformação aplicada a cada elemento do array original. Ele retorna um novo array com os elementos transformados.

array.map(callback(elemento, índice, array), thisArg) 
  
const numbers = [1, 2, 3, 4, 5];

const doubledNumbers = numbers.map(function(number) {
  return number * 2;
});

console.log(doubledNumbers);  // Output: [2, 4, 6, 8, 10]

Os métodos indexOf(), lastIndexOf(), includes(), find() e findIndex(), são usados para buscar elementos em um array

indexOf

O método indexOf() retorna o primeiro índice em que um determinado elemento é encontrado no array. Se o elemento não estiver presente, o método retorna -1.

array.indexOf(elemento, indiceInicial) 
        
const fruits = ["apple", "banana", "orange", "apple"];

console.log(fruits.indexOf("apple")); // Output: 0
console.log(fruits.indexOf("orange")); // Output: 2
console.log(fruits.indexOf("grape")); // Output: -1

lastIndexOf

O método lastIndexOf() é semelhante ao indexOf(), mas retorna o último índice em que um determinado elemento é encontrado no array. Novamente, se o elemento não estiver presente, o método retorna -1.

array.lastIndexOf(elemento, indiceInicial) 
  
const fruits = ['apple', 'banana', 'orange', 'apple', 'orange'];

console.log(fruits.lastIndexOf('apple'));  // Output: 3
console.log(fruits.lastIndexOf('orange'));  // Output: 4
console.log(fruits.lastIndexOf('grape'));  // Output: -1

includes

O método includes()verifica se um determinado elemento está presente no array e retorna true ou false com base nessa verificação.

array.includes(elemento, indiceInicial) 
  
const fruits = ["apple", "banana", "orange", "apple"];

console.log(fruits.includes("apple")); // Output: true
console.log(fruits.includes("grape")); // Output: false

find

O método find() retorna o primeiro elemento do array que satisfaz uma determinada condição especificada por uma função de callback.

array.find(callback(elemento, indice, array), thisArg) 
  
const numbers = [1, 2, 3, 4, 5];

const evenNumber = numbers.find(function (number) {
  return number > 2 ;
});

console.log(evenNumber); // Output: 3

findIndex

O método findIndex() é semelhante ao find(), mas retorna o índice do primeiro elemento que satisfaz uma determinada condição especificada por uma função de callback.

É especialmente útil quando você precisa encontrar o índice de um elemento específico em um array.

array.findIndex(callback(elemento, indice, array), thisArg) 
  
const fruits = ["apple", "banana", "orange"];

const index = fruits.findIndex(function (fruit) {
  return fruit === "banana";
});

console.log(index); // Output: 1, O resultado é o índice do elemento

Os métodos concat(), join() e toString(), que são usados para manipular e obter representações de strings a partir de arrays.

toString

array.toString() 
        
const fruits = ["apple", "banana", "orange", 3];

let arr1 =["a",2,3]

join

O método join() é usado para criar e retornar uma string que contém todos os elementos de um array, separados por um separador especificado.

array.join(separador) 
  
const numbers = [1, 2, 3];

const joinedString = numbers.join(" - ");
console.log(joinedString); // Output: 1 - 2 - 3

concat

O método concat() é usado para mesclar dois ou mais arrays, retornando um novo array resultante da concatenação.

array.concat(valor1, valor2, ..., valorN)

Os métodos push(), pop(), shift(), unshift(), slice() e splice(), que são utilizados para manipular arrays em JavaScript.

push

O método push()é usado para adicionar um ou mais elementos ao final de um array. Ele modifica o array original.

array.push(elemento1, elemento2, ..., elementoN) 
  
const fruits = ["apple", "banana"];

fruits.push("orange", "grape");
console.log(fruits); // Output: ['apple', 'banana', 'orange', 'grape']

pop

O método pop() é usado para remover o último elemento de um array e retorná-lo. Ele modifica o array original.

array.pop() 
  
const fruits = ['apple', 'banana', 'orange'];

const removedElement = fruits.pop();

console.log(removedElement);  // Output: 'orange'
console.log(fruits);  // Output: ['apple', 'banana']
  
  
//recuperar o elemento "removido"    

const exemplo = [1, 2, 3, 4, 5];

const removedElement = exemplo.pop();

console.log(removedElement); // Output: 5
console.log(exemplo); // Output: [1, 2, 3, 4]

shift

O método shift() é usado para remover o primeiro elemento de um array e retorná-lo. Ele modifica o array original.

array.shift() 
  
const fruits = ['apple', 'banana', 'orange'];

const removedElement = fruits.shift();

console.log(removedElement);  // Output: 'apple'
console.log(fruits);  // Output: ['banana', 'orange']

unshift

O método unshift() é usado para adicionar um ou mais elementos no início de um array. Ele modifica o array original.

array.unshift(elemento1, elemento2, ..., elementoN) 
  
const fruits = ['banana', 'orange'];

fruits.unshift('apple', 'grape');

console.log(fruits);  // Output: ['apple', 'grape', 'banana', 'orange']

slice

O método slice() é usado para "recortar" (shallow copy) de um subarray a partir de um array existente. Ele não modifica o array original.

array.slice(inicio, fim) 
  
const fruits = ['apple', 'banana', 'orange', 'grape', 'kiwi'];

const subarray = fruits.slice(1, 4);

console.log(subarray);  // Output: ['banana', 'orange', 'grape']

splice

O método splice() é usado para modificar um array, removendo, substituindo ou adicionando elementos a ele. Ele modifica o array original.

array.splice(inicio, quantidade, elemento1, elemento2, ..., elementoN) 
  
const fruits = ['apple', 'banana', 'orange', 'grape'];

fruits.splice(2, 1, 'kiwi', 'melon');

console.log(fruits);  // Output: ['apple', 'banana', 'kiwi', 'melon', 'grape']

  

  
  
const fruits = ['apple', 'banana', 'orange', 'grape'];

const removedElements = fruits.splice(1, 2);

console.log(removedElements); // Output: ['banana', 'orange']
console.log(fruits); // Output: ['apple', 'grape']

fruits.splice(1, 0, ...removedElements);

console.log(fruits); // Output: ['apple', 'banana', 'orange', 'grape']

reverse

O método reverse() é utilizado para inverter a ordem dos elementos em um array. Ele altera o array original e retorna uma referência para o array invertido.

  
const numbers = [1, 2, 3, 4, 5];
numbers.reverse();

console.log(numbers); // Output: [5, 4, 3, 2, 1]

reduce

O método reduce() é usado para reduzir os elementos de um array a um único valor. Ele executa uma função de callback em cada elemento do array, acumulando um valor resultante. O resultado final é retornado como o valor reduzido.

  
 const numbers = [1, 2, 3, 4];

let soma = numbers.reduce(function(accumulator, value) {
    return accumulator + value
})
console.log(soma)   
  

  
const nomes = ["Daniel", "Maria", "Joana", "João"]
let nomesPorOrdem = nomes.reduce(function (nomes, nomeAtual) {

    /*  1-
        nomes = {}
        nomeAtual = "Daniel" -> primeiraLetra = nomeAtual[0] = "D"
        return {D: 1}

        2-
        nomes = {D: 1}
        nomeAtual = "Maria" -> primeiraLetra = "M"
        return {D: 1, M: 1}

        2-
        nomes = {D: 1, M: 1}
        nomeAtual = "Joana" -> primeiraLetra = "J"
        return {D: 1, M: 1, J: 1}

        3-
        nomes = {D: 1, M: 1, J: 1}
        nomeAtual = "João" -> primeiraLetra = "J"
        return {D: 1, M: 1, J: 2}
    */

    let primeiraLetra = nomeAtual[0]
    if (nomes[primeiraLetra]) {
        nomes[primeiraLetra]++
    } else {
        nomes[primeiraLetra] = 1
    }
    return nomes
}, {})

console.log(nomesPorOrdem)    
  

  
const numeros = [1,3,4,1,4,5,3,5,8,9]

const numerosUnicos = numeros.reduce(function (numerosUnicos, numeroAtual) {
    if (numerosUnicos.indexOf(numeroAtual)< 0){
        numerosUnicos.push(numeroAtual)
    }
    return numerosUnicos
}, []);
console.log(numerosUnicos) // Output = [1,3,4,5,8,9]

spread

O operador spread é uma ferramenta poderosa para trabalhar com elementos iteráveis e pode facilitar a manipulação e combinação de dados de forma concisa

Ele permite que um elemento iterável, como um array ou uma string, seja expandido em lugares onde múltiplos elementos (ou argumentos) são esperados. O operador spread é representado por três pontos consecutivos ....

  
  //Expansao de uma array:

const array1 = [1, 2, 3];
const array2 = [...array1, 4, 5, 6];
console.log(array2); // Output: [1, 2, 3, 4, 5, 6]

 /* Nesse exemplo, o operador spread é usado para expandir os elementos do array1 em um novo array array2.
  Isso permite combinar facilmente os elementos de vários arrays em um único array. */
  

  
 //Passagem de argumentos para uma função:
 function sum(a, b, c) {
  return a + b + c;
}

const numbers = [1, 2, 3];
const result = sum(...numbers);
console.log(result); // Output: 6

/* Aqui, o operador spread é utilizado para passar os elementos do numbers como argumentos para a função sum(). 
Cada elemento é atribuído às respectivas posições dos parâmetros a, b e c, permitindo a execução da função com os valores corretos.*/
  

  
//Criação de cópias de objetos:

 const obj1 = { name: 'John', age: 30 };
const obj2 = { ...obj1 };
console.log(obj2); // Output: { name: 'John', age: 30 }
  
/* Nesse caso, o operador spread é usado para criar uma cópia superficial do objeto obj1 em obj2. 
Isso significa que obj2 terá as mesmas propriedades e valores que obj1, mas não compartilhará a mesma referência de memória. */

destructuring

A desestruturação de array permite que você extraia os elementos de um array e os atribua a variáveis separadas. 

  
const array = [1, 2, 3];

// Desestruturação de array
const [a, b, c] = array;

console.log(a); // Output: 1
console.log(b); // Output: 2
console.log(c); // Output: 3
  
  
const [n1, , n2] = array // , pulou o 2
console.log(n1) // 1
console.log(n2) // 3

Desestruturação de Objeto

A desestruturação de objeto permite que você extraia os valores das propriedades de um objeto e os atribua a variáveis separadas. É possível utilizar nomes diferentes para as variáveis, correspondendo às propriedades do objeto. Também é possível atribuir valores padrão a variáveis caso a propriedade correspondente não exista no objeto.

  
    const person = { name: 'Matheus', age: 100 };

// Desestruturação de objeto
const { name, age } = person;

console.log(name); // Output: 'Matheus'
console.log(age); // Output: 100

for...of

O for...of é uma estrutura de loop introduzida no JavaScript que permite iterar sobre elementos iteráveis, como arrays, strings, Map, Set, entre outros. Ele fornece uma maneira mais simples e concisa de percorrer os elementos de uma coleção, em comparação com o for tradicional.

for (variável of iterável) { // código a ser executado para cada elemento } 
  
const numbers = [1, 2, 3, 4, 5];

for (const number of numbers) {
  console.log(number); // Output: 1 2 3 4 5
}

  

  
 //O for...of também pode ser usado com strings:

 const message = "Hello";

for (const char of message) {
  console.log(char);
}
